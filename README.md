
## About the Project

This project was built using only vanilla HTML5 and CSS3 (no frameworks like Bootstrap or Tailwind, and no JavaScript). It focuses on demonstrating core front-end skills.

Key goals for this project included:

*   Creating reusable CSS classes and utility styles.
*   Practicing modern CSS layouts (e.g., Flexbox or Grid).
*   Ensuring the design is fully responsive for various screen sizes using media queries.
*   Utilizing CSS variables for a consistent and easily modifiable theme.

## Technologies Used

*   **HTML5
*   **CSS3**

## Getting Started

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository** to your local machine:
    ```bash
    git clone https://github.com/Rai321han/W3_HTML_CSS_ASSIGNMENT
    ```
2.  **Navigate** to the project directory:
    ```bash
    cd W3_HTML_CSS_ASSIGNMENT
    ```
3.  **Open** the `index.html` file in your preferred web browser to view the project.
